HTML元素

1、文档元素和元数据元素

用来创建HTML文档和说明其内容的元素

1.1、构筑基本的文档结构

  1. DOCTYPE元素：

    1. 在HTML5中DOCTYPE元素只有一种用法：<!DOCTYPE HTML>
    2. 这个元素告诉浏览器两件事：第一、它处理的是HTML文档。第二、用来标记文档内容的HTML所属的版本

  2. html元素：根元素，表示文档中HTML部分的开始。

  3. head元素：包含着文档的元数据，元数据向浏览器提供有关文档内容和标记的信息，此外还可以包含脚本和对外部资源（如CSS样式表）的引用。

    1. 每个HTML文档都应该包含一个head元素，head元素必须包含一个title元素。

  4. body元素：HTML文档的内容包装在body元素中，body元素总是紧跟在head元素之后，它是html元素的第二个子元素。

1.2、用元数据元素说明文档

元数据元素可以用来提供关于HTML文档的信息，它们本身不是文档内容，单提供了关于后面的文档内容的信息，元数据元素应该放在head元素中。

  1. title元素：title元素的作用是设置文档的标题或名称，浏览器通常将该元素的内容显示在其窗口顶部或标签页的标签上。

    1. 每个HTML文档都应该有且只有一个title元素。

  2. base元素：base元素可以用来设置一个基准URL，让HTML文档中的相对链接在此基础上进行解析。

    1. 相对链接省略了UTL中的协议、主机、端口部分，需要根据别的URL（要么是base中指定的URL，要么是用加载当前文档的URL）的出其完整形式。

    2. base元素还能设定链接在用户点击时的打开方式，以及提交表单时浏览器如何反映。

    3. HTML文档至少应该包含一个base元素，它通常是head元素中位置最靠前的子元素之一，以便随后的元数据元素中的相对URL可以用上其设置的基准URL。

1.4、使用脚本元素

与脚本相关的元素有两个：第一个是script，用于定义脚本并控制其执行过程。第二个是nioscript，用于规定在浏览器不支持脚本或禁用了脚本的情况下的处理办法。

  1. script元素：script元素可以用来页面中加入脚本，方式有在文档中定义脚本和引用外部文件中的脚本两种。最常用的脚本类型是JavaScript，不过浏览器也支持一些别的脚本语言。

  2. 这个元素所属类型因其用法而异，位于head元素中的script元素属于元数据元素，位于其他元素中的属于短语元素

2、文本层面元素

  1. a元素：生成超链接：

    a元素定义的6个局部属性如下图所示：

    1. 生成指向外部的超链接：将a元素的href属性设置为以http://开头的URL即可生成到其他HTML文档的超链接

    2. 使用相对URL：如果href属性值不是以http://这样的已知协议开关，那么浏览器会将该超链接视为相对引用，默认情况下，会假定目标资源与当前文档位于同一位置

    3. 生成内部超链接：超链接也可以用来将同一文档中的另一元素移入视野，为此需要用到形如CSS中针对目标元素的ID选择器的表达式：#<id>示例代码如下：

    4. target属性的用途是告诉浏览器希望将所链接的资源显示在哪里，默认情况下浏览器使用的是显示当前文档的窗口，target属性值如下图所示：

  2. b元素可以用来标记一段文字，但并不表示强调或重要性。

    b元素的作用非常简单：它的开始标签和结束标签之间的内容会从周围内容中凸显出来，通常是将其内容加粗。

  3. em元素表示对一段文字的强调。

  4. i元素表示一段文字与周围内容有本质区别

  5. s元素用来表示一段文字不再正确或准确，其习惯样式是在文字上显示一条删除线

  6. strong元素表示一段重要文字。

  7. u元素让一段文字从周围内容中凸显出来，但并不代表强调或重要性有所增加，效果为添加下划线

  8. small元素表示小号字体内容，常用于免责声明和澄清声明

  9. sub和sup元素分别用于表示上标和下标

  10. br和wbr元素：

    1. br是强制换行，其习惯样式是将后续内容转移到新行上

    2. wbr元素用来表示长度超过当前浏览器窗口的内容适合在此换行。

    示例代码如下：

    Code

  12. abbr元素用来表示缩写，其title属性表示的是该缩写代表的完整词语

  13. dfn元素表示定义中的术语，也即在用来解释一个词，或短语含义的句子中的词或短语

  14. q元素表示引自他处的内容

  15. cite元素表示所引用作品（如图书、文章、电影、诗歌）的标题。

  16. 语言符号：ruby元素表示一段包含注音符号的文字，ruby元素需要与rt元素和rp元素一起使用，rt元素用来标记注音符号，rp元素则用来标记不支持注音符号特性的浏览器显示在注音符号前后的括号。

  17. bdo元素可以用来撇开默认的文字方向设置，明确的指明其内容中文字的方向，bdo元素必须加上dir属性，该属性有rtl：从右到左，ltr：从左到右两个值

  18. bdi元素表示一段出于文字方向考虑而其他内容隔离开来的文字

  19. span元素本身没有任何意义，它可以用来把一些全局属性应用到一段内容上。可以用来配置CSS

  20. mark元素是HTML5新增的，用来表示因为某段上下文相关而被突出显示的一段文字
