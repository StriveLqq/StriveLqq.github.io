一、hexo常用命令

1、创建新的博客

hexo new "博客名"

hexo n "博客名"

参数

-p, --path ：自定义文件目录

-r, --replace：如果存在同名文章，将其替换

-s, --slug：文章的slug，作为新文章的文件名和发布后的url

2、清理缓存文件和已生成的静态文件

hexo clean

3、生成静态文件

hexo generate

hexo g

参数

-d, --deploy：文件生成后立即部署网站

-w, --watch：监视文件变动

-b, --bail：生成过程中如果发生任何未处理的异常则抛出异常

-f, --force：强制重新生成文件，住：未加参数时，如果 public 目录存在，那么 hexo g 只会重新生成改动的文件

-c, --concurrency：最大同时生成文件数量，默认为无限制

4、启动服务器

hexo server

hexo s

参数

-p, --port：重设端口

-s, --static：只使用静态文件

-l, --log：启动日记记录，使用覆盖记录格式

5、部署网站

hexo deploy

hexo d

参数

-g, --generate：部署之前预先生成静态文件
